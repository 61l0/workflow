== Description ==
Provide a Workflow Engine, Design, Prototyping and Execution of Workflow totally from WordPress

**Features include:**

* **Model:** Web Based Business Process Modeler supporting BPMN 2.0; all from your browser, no tools to install

* **Design:** Defines Business Process implementation like flow conditions, Business Rules for Data Validation, Access Rules in addition to BPMN Validation

* **Test:** Fully simulate the runtime environment for various participants and validate the design.

* **Execute:** End users can access your process through Front-end dashboard, Interact with the process flow by providing information and manage Task Assignments

* **Monitor:** In addition Omni-Workflow offers a Monitor allowing authorized users to keep a watch on your business.

* **Full Business Application in a Box:** In short, offer full customizable business application in a box.

== Installation ==

1. Install either via the WordPress.org plugin directory, or by uploading the files to your server.
2. Activate the plugin
3. Setup a menu option for your front end users to access Workflow processes
4. Configure User roles by adding Omni Workflow Capabilities to your users (optional)
5. That's it.  You're ready to go!
